Contributing
------------

0\. Install Cabal and get to the latest version, to have sandboxes

```
    # OSX
    $ brew install haskell-platform
    $ cabal install cabal-install --global
```

1\. Set up a cabal sandbox for this project

```
    $ cabal sandbox init
```

2\. Configure and build

```
    $ cabal configure
    $ cabal build
```
